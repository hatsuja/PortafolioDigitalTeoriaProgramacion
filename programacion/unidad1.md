# <span style="color:#2c3e50">Algoritmo</span>

## <span style="color:#2980b9">¿Qué es un algoritmo?</span>
Un algoritmo es básicamente una receta o una lista de pasos que seguimos para resolver un problema o completar una tarea.

**Ejemplo: Preparar un café**
1. Pones agua a hervir.
2. Colocas el café en el filtro.
3. Viertes el agua caliente.
4. Sirves en una taza.

### <span style="color:#16a085">Características principales:</span>
* **Es ordenado:** Los pasos van uno tras otro.
* **Es preciso:** Cada paso es claro y no da lugar a dudas.
* **Tiene un final:** La tarea se termina.

---

## <span style="color:#2980b9">Clasificación de un algoritmo</span>

### <span style="color:#16a085">Cualitativos</span>
Son aquellos que usamos con palabras, como una receta o instrucciones de armado.
* **Ejemplo:** Cambiar un foco (Apagar luz, subir escalera, girar bombilla...).

### <span style="color:#16a085">Cuantitativos</span>
Son los que utilizan cálculos numéricos.
* **Ejemplo:** Calcular el área de un triángulo ($A = \frac{b \times h}{2}$).

---

## <span style="color:#2980b9">Estructura de un algoritmo</span>
1. **Entrada (Input):** Datos iniciales (ej. año de nacimiento).
2. **Proceso:** Operaciones de transformación (ej. la resta para sacar la edad).
3. **Salida (Output):** Resultado final (ej. la edad calculada).

---

## <span style="color:#2980b9">Representación y Pseudocódigo</span>
Se puede representar en **Lenguaje Natural** (nuestro idioma) o **Pseudocódigo** (un punto medio técnico).

**Ejemplo en PSeInt:**
```pseint
Algoritmo CalcularPromedio
    Escribir "Ingresa la nota:"
    Leer nota
    promedio <- nota / 1
    Escribir "Tu promedio es: ", promedio
FinAlgoritmo
