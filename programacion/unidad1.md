# <span style="color:#2c3e50">Algoritmo</span>[cite: 1]

## <span style="color:#2980b9">¿Que es un algoritmo?</span>[cite: 1]
Un algoritmo es básicamente una receta o una lista de pasos que seguimos para resolver un problema o completar una tarea.[cite: 1]

**Por ejemplo, preparar un café:**[cite: 1]
1. Pones agua a hervir.[cite: 1]
2. Colocas el café en el filtro.[cite: 1]
3. Viertes el agua caliente.[cite: 1]
4. Sirves en una taza.[cite: 1]

### <span style="color:#16a085">Características principales:</span>[cite: 1]
* **Es ordenado:** Los pasos van uno tras otro.[cite: 1]
* **Es preciso:** Cada paso es claro y no da lugar a dudas.[cite: 1]
* **Tiene un final:** la tarea se termina.[cite: 1]

---

## <span style="color:#2980b9">Clasificación de un algoritmo</span>[cite: 1]

### <span style="color:#16a085">Cualitativos</span>[cite: 1]
Son aquellos que usamos con palabras, como una receta de cocina o las instrucciones para armar un mueble.[cite: 1]

**Ejemplo: Cambiar una bombilla foco**[cite: 1]
* Apagar el interruptor de la luz.[cite: 1]
* Colocar una escalera debajo de la lámpara.[cite: 1]
* Subir la escalera con cuidado.[cite: 1]
* Girar la bombilla vieja hacia la izquierda hasta que salga.[cite: 1]
* Colocar la bombilla nueva y girarla hacia la derecha hasta que ajuste.[cite: 1]
* Bajar de la escalera y encender el interruptor.[cite: 1]

### <span style="color:#16a085">Cuantitativos</span>[cite: 1]
Son los que utilizan cálculos numéricos y matemáticas para llegar a un resultado.[cite: 1]

**Ejemplo: Calcular el área de un triángulo**[cite: 1]
1. Conocer la medida de la base (b).[cite: 1]
2. Conocer la medida de la altura (h).[cite: 1]
3. Multiplicar la base por la altura (b*h).[cite: 1]
4. Dividir el resultado anterior entre 2.[cite: 1]
5. El resultado final es el área (A).[cite: 1]

En matemáticas se vería así:  
$$A = \frac{b \times h}{2}$$[cite: 1]

---

## <span style="color:#2980b9">Estructura de un algoritmo</span>[cite: 1]
Un algoritmo de compone de 3 partes:[cite: 1]

1. **Entrada (Input):** Información o datos iniciales.[cite: 1]
2. **Proceso:** Conjunto de pasos, cálculos u operaciones.[cite: 1]
3. **Salida (Output):** Resultado final que se obtiene.[cite: 1]

---

## <span style="color:#2980b9">Representación de un algoritmo</span>[cite: 1]

### <span style="color:#16a085">1. Lenguaje Natural</span>[cite: 1]
Uso de nuestro propio idioma.[cite: 1]

### <span style="color:#16a085">2. Pseudocódigo</span>[cite: 1]
Punto medio entre idioma y programación.[cite: 1]

**Estructura básica de PSeInt:**[cite: 1]
```cpp
Algoritmo NombreDelPrograma
    // Aquí van las declaraciones y procesos
FinAlgoritmo
```[cite: 1]

### <span style="color:#16a085">Ejemplo básico completo:</span>[cite: 1]
```cpp
Algoritmo CalcularPromedio
    // 1. Entrada de datos
    Escribir "Por favor, ingresa la primera nota:"
    Leer nota1
    Escribir "Ingresa la segunda nota:"
    Leer nota2
    
    // 2. Proceso (Cálculo)
    promedio <- (nota1 + nota2) / 2
    
    // 3. Salida de resultados
    Escribir "Tu promedio final es: ", promedio
    
    Si promedio >= 7 Entonces
        Escribir "¡Felicidades! Has aprobado."
    Sino
        Escribir "No te desanimes, sigue practicando."
    FinSi
FinAlgoritmo
```[cite: 1]

## <span style="color:#2980b9">Diagrama de flujos</span>[cite: 1]
* **Óvalo:** Inicio/Fin.[cite: 1]
* **Paralelogramo:** Entrada/Salida.[cite: 1]
* **Rectángulo:** Proceso.[cite: 1]
* **Rombo:** Decisión (Sí/No).[cite: 1]
* **Flechas:** Líneas de flujo.[cite: 1]

## <span style="color:#2980b9">Prueba de escritorio</span>[cite: 1]
Se utiliza para verificar si un algoritmo funciona correctamente de forma manual.[cite: 1]
