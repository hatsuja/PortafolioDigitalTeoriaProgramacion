#

## ¿Qué es un Lenguaje de Programación?
Es un conjunto de símbolos y reglas que permiten escribir programas para que la computadora realice acciones específicas

---

## Clasificación de los Lenguajes de programacion

<div align="center">
  
| Lenguaje | Características | Ejemplos / Notas |
| :--- | :--- | :--- |
| **Lenguaje Máquina** | Instrucciones directas al procesador | Código binario ( `0` y `1` ). |
| **Bajo Nivel (Assembler)** | Utiliza abreviaturas para representar instrucciones | Mnemotécnicos como `ADD` o `DIV`. |
| **Alto Nivel** | Diseñados para que las personas los entiendan fácilmente | Palabras en inglés (ej. `C++`, `Java`, `Python` ). |
| **Algorítmico** | Uso de lenguaje natural para describir procesos | Español o Pseudocódigo. |

</div>

---

## lenguaje Compilafos e Interpretados

<div align="center">

| Característica | Lenguajes Compilados | Lenguajes Interpretados |
| :--- | :--- | :--- |
| **Traducción** | Traduce todo el código de una sola vez mediante un programa llamado compilador | Traduce el código línea por línea mientras el programa se está ejecutando |
| **Resultado** | Genera un archivo ejecutable independiente | No genera un ejecutable; requiere del intérprete para funcionar |
| **Ventaja** | Son más rápidos al ejecutarse | Son más flexibles ante cambios en el código |
| **Desventaja** | Son menos flexibles una vez compilados | Suelen ser más lentos que los compilados |
| **Ejemplos** | `C` , `C++` | `Python` , `JavaScript` |

</div>
